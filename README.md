Инструкция:

1. Нажать кнопку "Выбрать файл" для загрузки фото.
2. Добавить отрезки, которые необходимо рассчитать.
3. Ввести размер первого отрезка в поле "Введите число".
4. Нажать "Рассчитать значения".
